# Mini-storefront 

This is a mini store front project, where users can add and take away items from a cart.

Set up: To set up 
1. Open terminal
2. Type npx create-next-app@latest mini-storefront
3. Type Run npm run dev
4. Open http://localhost:3000

This project includes:
Cart feature that allows the user to add items
Cart feature that allows the user to take away items
A filter that lets the user filter out categories
Allows the user to set a Max price
